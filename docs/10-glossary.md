# Glossary

## Prompt
The user message sent to the model.

## System Prompt / System Instruction
Hidden instructions that define assistant behavior.

## Guardrail
A safety rule that constrains model behavior.

## Suite
A group of test cases.

## Case
One test scenario with prompt + expected outcome.

## Severity
How serious a violation is (`critical`, `high`, `medium`, `low`, `info`).

## Threshold
Limits that decide pass/fail for counts by severity.

## Violation
A failed rule check.

## Artifact
A saved output file from a run (`report.json`, `report.junit.xml`, `report.html`).

## Allowlist
Explicit list of tools that are allowed.

## Prompt Injection
Input that tries to override safety instructions.

## Exfiltration
Attempts to pull private/sensitive data.
